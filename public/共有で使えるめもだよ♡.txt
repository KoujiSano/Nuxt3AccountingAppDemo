✨Publicにある画像の入れ方
src="/hoge.png"
パスが簡単ですごーい🙆🏻‍♂️

✨サイトにタイトルをつける方法
<script setup lang="ts">
useHead({ title: 'にゃんにゃん～' })
</script>
これを一番上に入れればおけ～🤩

✨composablesを使ってページ遷移後にも情報を保持する方法
1.composablesの中に"DarkSide".tsを作成して関数を作ろう！
export const "OtakuMoe" = () => {
    return useState<any>('data', () => "なーん..にゃん")
  }

(🧨オブジェクトを使う場合)
export const "OtakuMoe" = () => {
    return useState('User', () => shallowRef({
        Cloe: "＼(^o^)／",
        Mona: "オマエモナー",
    }))
}

2.vueファイルのscriptに下記を追加！
<script setup lang="ts">
import { OtakuMoe } from '@/composables/"DarkSide"'
const "hoge" = OtakuMoe()
</script>

3.表示させてみよう～
<p>{{ hoge }}<p>
(🧨オブジェクトを使う場合)
<p>{{ hoge.Cloe }}<p>

(これを使えばどこでもユーザー情報を表示できるようになるかも😁)

  <b-option value="たけのこ"> たけのこ </b-option >
